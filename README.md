# colombian_salary
🇨🇴 Interactive 2026 Colombian payroll guide for international teams. Covers ordinary &amp; integral salary, social security, parafiscal contributions, Law 1607 exemptions, and a full employer cost calculator. Built with HTML5, CSS3 &amp; vanilla JS — no dependencies.

A single-file interactive web guide built with HTML5, CSS3 & vanilla JavaScript to help European and international managers understand Colombia's payroll system. Covers the 2026 SMMLV values, ordinary vs. integral salary structures, social security obligations (EPS, AFP, ARL), parafiscal contributions, Law 1607/Decree 114-1 exemptions, and an employer cost calculator supporting both salary types.
